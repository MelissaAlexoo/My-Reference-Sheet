CSS Selectors

CSS Selector Name Description
h1/ h2/ h3 Type This is often referred to by the opening tag name or element selector.
.box Class This allows us to style two classes at once by using a full stop before the element name.
#unique ID This only styles one element and also override the styles of types of classes.

Attribute Selectors

Attribute Selector Name Description
[href] HRef This targets all elements with a href attribute and sets rules accordingly.
src Source This targets each media example on website accordingly.
